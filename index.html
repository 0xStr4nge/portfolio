About

I am a passionate cybersecurity practitioner focused on penetration testing and vulnerability research. I enjoy breaking down systems to understand how they can fail — then building clear, practical fixes to make them safer. My approach mixes methodical reconnaissance, hands-on exploitation, and careful reporting so teams can act quickly and confidently.

I sharpen my skills through targeted labs, CTF challenges, and active participation in bug bounty programs (HackerOne & Bugcrowd). Technically, I work across web application security, network penetration, cloud basics, and Active Directory environments — using tools like Burp Suite, Nmap, Wireshark, and Metasploit while scripting in Python when needed.

I value clear communication and reproducible findings: every report I produce includes concise steps-to-reproduce, risk prioritization, and remediation guidance. I’m always learning new techniques and collaborating to improve security posture.
